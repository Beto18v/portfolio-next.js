# Portfolio Next.js

Portfolio personal desarrollado con Next.js, TypeScript y Tailwind CSS.

## Características principales

- Interfaz moderna y responsiva
- Sección de proyectos destacados
- Habilidades técnicas
- Formulario de contacto
- Soporte para temas claro/oscuro
- Internacionalización (multi-idioma)

## Tecnologías utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/portfolio-next.js.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Despliegue

Puedes desplegar este portafolio fácilmente en Vercel, Netlify u otras plataformas compatibles con Next.js.
